# WitShells — Reusable Unity Scripts & Packages

A collection of production-ready, battle-tested design patterns, utilities, and gameplay systems authored for Unity.

<div class="cards">

## 🏗️ Design Patterns
Generic, engine-agnostic implementations of the most common software design patterns — Observer, State Machine, Object Pool, Command, Builder, Service Locator, and more.  
[View API →](api/WitShells.DesignPatterns.Core.html)

## 🎮 Gameplay Systems
Ready-made modules for third-person character control, tank controls, shooting system, vehicle control, turn-based logic, and formation utilities.

## 🖼️ UI Toolkits
Canvas draw tool, MCQ UI, dialog manager, and a spiral layout container — all drag-and-drop ready for your Unity project.

## 🌐 Networking
WebSocket and WebRTC integrations, multiplayer scaffolding, and a lightweight API integration layer.

</div>

## Quick Start

Install any package via Unity's **Package Manager → Add package from git URL**:

```
https://github.com/your-username/Reusable-Unity-Scripts-Packages.git?path=Assets/WitShells/DesignPatterns
```
